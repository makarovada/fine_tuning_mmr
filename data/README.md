# Описание набора данных

## Задача
Классификация изображений на три класса животных: медведь (bear), панда (panda), енот (raccoon). Набор данных создан для демонстрации transfer learning на небольшом наборе данных.

## Метод сбора
Изображения собраны вручную из открытых источников (Google Images, Unsplash, Wikimedia Commons). Каждое изображение проверено на соответствие классу. Все изображения находятся в общественном достоянии или под свободной лицензией (CC0, CC BY-SA).

## Классы
- **Медведь (bear)**: Изображения различных видов медведей (бурый, черный, белый) в естественной среде или зоопарках.
- **Панда (panda)**: Изображения больших панд, преимущественно в зоопарках или заповедниках.
- **Енот (raccoon)**: Изображения енотов-полоскунов в естественной среде или городских условиях.

## Статистика
- Общее количество изображений: 135 (bear: 46, panda: 44, raccoon: 45).
- Разделение: 70% train (~94 изображения), 15% validation (~20), 15% test (~21).

## Предобработка
- Ресайз: Изображения приводятся к размеру 224x224 (для совместимости с моделями ResNet/EfficientNet).
- Нормализация: Средние значения [0.485, 0.456, 0.406], стандартное отклонение [0.229, 0.224, 0.225] (ImageNet).
- Аугментации (для train): Random Horizontal Flip (p=0.5), Random Rotation (±15°), Color Jitter (яркость/контрастность), Random Crop.
- Для val/test: только ресайз и нормализация.

## Структура
- `data/raw/`: Необработанные изображения, разделенные по классам.
- `data/split/`: Разделенные данные (train/val/test).
